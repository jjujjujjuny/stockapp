  USD 자산 처리 (질문 답변)

  달러 자산도 완전히 지원해요:
  - 설정 탭에서 각 종목의 통화를 원화(KRW) / 달러(USD) 선택 가능
  - 달러 자산은 GAS 서버에서 Naver Finance 환율 API로 실시간 USD/KRW 조회 후 원화 환산
  - 리밸런싱 계산 시 원화 기준으로 통일해서 "몇 주 살지" 계산
  - 화면에는 $XX.XX (≈XXX만원) 형태로 표시
  - 앱 상단에 현재 환율 바 표시 (USD 자산 있을 때만)

  설정 방법 (순서대로)

  1단계 — Google Sheets 준비
  1. 구글 드라이브에서 새 스프레드시트 생성
  2. Extensions > Apps Script 열기
  3. Code.gs 내용 전체 붙여넣기
  4. 상단에서 함수 선택 → initSheets → ▶ 실행 (최초 1회)
  → Holdings, Config 시트 자동 생성

  2단계 — GAS 웹앱 배포
  1. Deploy > New deployment > Web App
  2. Execute as: Me / Who has access: Anyone
  3. 배포 URL 복사 (.../exec 형태)

  3단계 — 앱 연결
  1. index.html을 GitHub에 올리고 GitHub Pages 활성화
  2. 앱 열기 → ⚙️ 설정 탭 → GAS URL 붙여넣기 → 저장

  4단계 — 배당금 설정
  - 설정 탭에서 각 ETF의 주당 배당금(원)과 지급월 체크

  ▎ 현재 포트폴리오 5종목(0072R0, 379810, 379800, 441640, 458730) 모두 원화 ETF라 환율 변환은 달러 자산 추가 시에만
  작동해요.
